Purpose

The bootstrap-modal-carousel.js library is made to display a carousel in a fullscreen modal box. The slides fit automatically.

The script depends on bootstrap-modal-fullscreen, bootstrap-modal-local and bootstrap-carousel-fit libraries.

Code

  1. Add the class name .modal-fullscreen to the modal element to display it in fullscreen.
  2. Add the attribute data-modalize="true" to the carousel element to add some features with fullscreen.
  3. Add the class name .carousel-fit to the carousel element to have the caption fitted.
  4. You can also add one of the three class names .carousel-caption-active, .carousel-caption-hover or .carousel-caption-hide to the carousel element to add some effects to the captions.
<div id="myCarousel" class="carousel slide carousel-fit carousel-caption-active"  data-modalize="true">
	<!-- Carousel items -->
	<div class="carousel-inner"></div>
	<!-- Carousel nav -->
	<a class="carousel-control left" href="#myCarousel" data-slide="prev">&lsaquo;</a>
	<a class="carousel-control right" href="#myCarousel" data-slide="next">&rsaquo;</a>
</div>

<div id="myModal" class="modal hide fade modal-fullscreen" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
	<div class="modal-header">
		<button type="button" class="btn btn-danger" data-dismiss="modal" aria-hidden="true">
			<i class="icon-remove icon-white"></i>
			<span>Close</span>
		</button>
	</div>
	<div class="modal-body">
		<p>One fine body…</p>
	</div>
</div>
